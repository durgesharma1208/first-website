🚍 Routify

Routify is a travel booking platform that allows users to search and book buses, trains, and flights from a single interface.
The goal of Routify is to simplify transportation booking by combining multiple travel services into one platform.

🌐 Live Demo

🔗 Website:
[https://your-deployment-link.com](https://travel-frontend-l7vs.onrender.com/)

📌 Features

🚌 Bus Booking

Search buses between cities

View available schedules

Book seats easily

✈ Flight Booking

Search flights between destinations

View pricing and timing

Quick booking interface

🚆 Train Booking

Train availability lookup

Schedule and seat details

Booking functionality

🔎 Unified Search

Search bus, train, and flight routes in one place

📱 Responsive Design

Works on mobile, tablet, and desktop devices

⚙️ Installation

Run the project locally:

# clone repository
git clone https://github.com/durgesharma1208/routify.git

# enter project folder
cd routify

# install dependencies
npm install

# start server
npm start
